# 码程 · Codeworks

面向**软件/IT 外包公司**的多租户 SaaS —— 专业服务自动化（PSA）+ 轻 ERP，以"**项目全生命周期 → 工时 → 实时盈亏**"为核心闭环。

- **目标用户**：软件/IT 外包公司（中小企业 10–200 人）
- **形态**：中文、多租户、按席位订阅、云 SaaS（Web）
- **技术栈**：TypeScript 全栈 · NestJS · PostgreSQL · Prisma · React · Tailwind + shadcn/ui
- **架构**：模块化单体（modular monolith）+ pnpm monorepo

> 📄 完整设计规格见 [`docs/superpowers/specs/2026-07-04-codeworks-psa-erp-design.md`](docs/superpowers/specs/2026-07-04-codeworks-psa-erp-design.md)

## 路线图（三阶段）

- **Phase 1 · MVP**：项目盈亏闭环（立项→Scrum 执行→工时→成本→实时盈亏）
- **Phase 2 · 商业化与钱流**：完整权限/CRM/报价合同计费/工作流引擎/客户门户/报表
- **Phase 3 · 深化与生态**：增值税票·在线支付·IM/研发工具集成·运维续约·对象存储
